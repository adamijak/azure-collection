# Azure HTTP
This repository is collection of HTTP requests to Microsoft Azure and O365 services.
It is based on .http file format

## Installation
Clone this repository:
```sh
git clone git@github.com:adamijak/azure-collection.git
```

## Usage
To use this collection open it with your favourite HTTP client.
You can use Visual Studio Code with REST Client extension, Jetbrains IDEs, Visual Studio or any other CLI tool.

Make sure you fill the `token` variable with your access token.
To obtain the token you can use [azure-auth](https://github.com/adamijak/azure-auth).
